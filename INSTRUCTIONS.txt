STEP1) DOWNLOAD THE ZIP FOLDER FROM GITHUB
STEP2) EXTRACT THE FOLDER FROM THE ZIP.
STEP3) OPEN COMMAND PROMPT AND CD INTO THIS FOLDER. THEN RUN THE FOLLOWING COMMANDS:

npm run build 
npm run StartRehnuma

THIS WILL RUN THE WEB APP ON LOCAL HOST. 
OPEN THIS LOCAL HOST ON ANY WEB BROWSER TO SEE TE WEB APP.


IF YOU ARE THE ADMIN AND WANT TO UPDATE THE DATABASE OF SUPERMARKETS, THEN PERFORM:
npm run ScrapeData
and once the csvs are generated do:
npm run UpdateDatabase
